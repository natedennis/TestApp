Welcome to Monarch Mobile Demo!

It runs on a clusterable JBoss AS 7 application server. This project leverages nothing but open source technology in order to reduce cost and retain full control of source free from the license fees associated with proprietary technologies. It is built using Java EE 6, Jboss Seam, Richfaces 4.3 Snapshot, Hibernate and HTML5.
